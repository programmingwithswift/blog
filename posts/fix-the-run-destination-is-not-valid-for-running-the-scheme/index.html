Fix "the Run Destination Is Not Valid for Running the Scheme"

This is can be a frustrating issue. One of the fixes for this is to restart Xcode. There can also be another solution and that is to change the device that you have chosen for deployment.

This issue can also occur if you are trying to run the app on an iPhone but you have set your Device under Deployment info to iPad, or vice versa.
